# Scripts

Personal showcase site for small Python utility tools, built with Jekyll and hosted on GitHub Pages.

Visit: **[Shimin-CUFE.github.io](https://Shimin-CUFE.github.io)**

## What's on the site

**ExcelTools** — Batch Excel search and merge. Search thousands of `.xls` / `.xlsx` files for a keyword in seconds, or merge scattered spreadsheets into one clean master table with automatic header alignment.

**img2pdf** — Image to PDF converter. Drag in a folder of photos, reorder and preview, then export a single A4 PDF with auto-extracted EXIF date captions. Pure Tkinter, single-file app.

**Blackjack** — Classic 21-point card game. Full casino rules (split, double, insurance), chip betting, and a green-felt card table UI. Also comes with a retro console mode using Unicode suit symbols.
